# Objetivos do Projeto

Este projeto pretende estabelecer um sistema robusto de documentação partilhada para centralizar e organizar informação de forma colaborativa, eficiente e produtiva.

## Metas
- Criar estrutura clara
- Facilitar colaboração
- Garantir rastreabilidade de alterações
- Melhorar comunicação entre equipa
- Implementar controlo de qualidade
- Automatizar processos de validação
